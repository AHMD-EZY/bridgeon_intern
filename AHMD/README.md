# AHMD - Python Movie Booking Starter

This folder contains a minimal Python starter for a movie booking demo using SQLite.

Files:
- `main.py` - initializes `movies.db` and seeds sample movies if empty.

Quick start (PowerShell):

```powershell
cd d:\vscode\AHMD
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python main.py
```

The script creates `movies.db` in the same folder and prints "Initialized movies.db".
